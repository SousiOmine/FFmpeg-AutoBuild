このリポジトリは、FFmpegをGitHub workflowを用いてビルドし、GitHub Releaseでビルド成果物を配布するワークフロー定義を格納したものです。
READMEおよびワークフローはMITライセンスにてライセンスされますが、これはビルド成果物であるFFmpegのライセンスを上書きするものではありません。

Releaseにて配布されるソフトウェアは、[FFmpeg](http://ffmpeg.org) のコードを使用しており、
[LGPL v2.1](https://www.gnu.org/licenses/old-licenses/lgpl-2.1.html) ライセンスに基づいて提供されています。
ソースコードは [こちら](https://github.com/SousiOmine/FFmpeg-AutoBuild/releases) からダウンロードできます。
